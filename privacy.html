Clip-It — Privacy Policy

Effective: May 26, 2026

1. Scope

Clip-It is a self-hosted, single-user application installed and run by its operator on their own computer. It does not collect, transmit, or share personal data with the developer or any third party other than the platforms the user explicitly authenticates with.

2. Data handled locally

Clip-It stores the following data only on the user's own machine:

  • YouTube channel URLs and video metadata for creators the user chooses to track
  • Locally downloaded video files
  • OAuth refresh tokens for connected services (e.g., TikTok), stored in a local file accessible only to the user

No data is sent to any server controlled by the developer of Clip-It.

3. TikTok integration

When the user connects their TikTok account via OAuth, Clip-It receives an access token and refresh token from TikTok. These tokens are stored locally and used only to upload videos to the authenticated account at the user's direction. Tokens are never shared with anyone. The user may revoke access at any time from their TikTok account settings or by disconnecting in Clip-It.

4. Scopes requested

Clip-It requests the following TikTok scopes:

  • user.info.basic — read basic profile information of the connected account
  • video.upload — upload videos to the connected account's drafts/inbox for the user to publish manually

5. Data retention

All data remains on the user's local machine until the user deletes it. Uninstalling Clip-It and deleting its folder removes all stored data.

6. Children

Clip-It is not directed at children under 13 and does not knowingly collect any data from them.

7. Contact

For privacy questions about Clip-It, contact the operator running the application.
